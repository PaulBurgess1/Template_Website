# Template_Website

This project uses:
  * Bootstrap, jQuery
  * Font Awesome
  * Images from unsplash.com

Current Template pages:
  * Index/Home
  * Login
  * Registration 

Future features:

  * More template pages
     * About Us, Contact Us, Pricing, Shopping-Cart, Reviews
  * More Web Accessibility. Colors, Fonts
